# React + Vite

## Preview

![Preview](./preview-news-api.png)

## Description

This is a React + Vite project that uses the News API to display news articles. 

## Usage

```bash
pnpm install

# or

yarn install

# or

npm install
```

## Run

```bash
pnpm dev

# or

yarn dev

# or

npm run dev
```

## Features

- [x] React
- [x] Vite
- [x] Axios
- [x] News API
- [x] ESLint
